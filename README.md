# Amadeus Project

Amadeus is a high-performance, plugin-based messaging core written in Rust.

## Documentation

- [Architecture Overview](amadeus/doc/ARCHITECTURE.md) - Deep dive into the core system design.
- [Enterprise Integration Guide](README_INTEGRATION.md) - Guide for the Web (Vue) and Java (Spring Boot) integration layer.
- [Developer Guide](amadeus/doc/DEVELOPER_GUIDE.md) - How to write plugins and contribute.

## Project Structure

- `amadeus/`: The Rust Core.
- `amadeus-server/`: Spring Boot Middleware.
- `amadeus-web/`: Vue 3 Frontend.
